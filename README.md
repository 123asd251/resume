🛸 ZGH's Portfolio | 技术沉淀与工程实践
欢迎来到我的个人技术空间。这里不仅是一份在线简历，更是我作为全栈工程师在过去五年中的技术思考、架构演进及复杂问题解决方案的深度记录。

💡 建设初衷
在软件开发领域，代码是最终产出，而文档是思考的足迹。我搭建这个站点的目的是：

深度复盘：对 X-Hub、ZenNote 等复杂项目进行模块化拆解，记录从技术选型到落地的完整路径。

知识体系化：将零散的 Java (Spring Boot)、Rust (Tauri)、Vue 3 经验构建成可索引的知识库，助力面试与职业进阶。

工程化展示：通过 VitePress + GitHub Actions 的全自动化部署实践，展示对现代前端工程化的理解。

🏗️ 核心模块规划
1. 项目经验复盘 (Project Deep-Dive)
X-Hub 企业级插件化平台：深度探讨基于 PF4J 的动态加载机制，如何解决类加载隔离与插件热插拔难题。

ZenNote 跨平台笔记：记录从 Electron 到 Tauri 的迁移过程，分享 Rust 与 JS 之间高效 IPC 通信的实战经验。

GIS 专项研究：针对 Raster 数据处理 与 TiTiler 的集成实践，记录在处理超大 GeoTIFF 时的性能优化方案。

2. 技术深挖 (Technical Insights)
后端演进：Spring Boot 3 响应式架构与复杂业务场景下的设计模式应用。

前端基石：Vue 3 组合式 API 的逻辑封装实践及 Vite 构建性能调优。

系统底层：Rust 内存安全模型在桌面端开发中的优势分析。

3. 面试攻略 (Interview Preparation)
核心算法、设计模式的个人化解读，拒绝死记硬背，强调场景化应用。

🛠️ 技术实现
本站采用业界领先的静态网站生成方案：

框架: VitePress (Vue-powered SSG)

部署: GitHub Pages

CI/CD: GitHub Actions (自动化构建与发布)

渲染引擎: Vite (极速冷启动与热更新)
